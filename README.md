# renewable-energy-plant-sys

## Prerequariments:
- JDK 11+
- sbt
- Scala 2.13.14

## How to run the project in Visual Studio
1. Install "Scala (Metals) extentions

2. Clone the repository
git clone https://github.com/Mordekay15/renewable-energy-plant-sys.git

cd renewable-energy-plant-sys

3. Metals will show a notification: "Import build" → click Import build

4. Wait ~2 minutes while it downloads Scala + dependencies (watch the status bar at the bottom)

5. Run
sbt run
